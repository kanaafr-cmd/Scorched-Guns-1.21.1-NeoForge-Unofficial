# Scorched-Guns-1.21.1-NeoForge-Unofficial# Scorched Guns 2 — Unofficial NeoForge 1.21.1 Port

Unofficial port of Scorched Guns 2 to NeoForge 1.21.1.

This project is experimental and not affiliated with the original developers.

## Status

Current state:
- Core gameplay functional
- GeckoLib gun animations partially restored
- Most weapons/items load correctly
- Active bug fixing and polish ongoing

Known issues:
- Some reload synchronization bugs still exist
- Some third-person gun animations/poses still need work
- Some recipes may still be broken
- Multiplayer testing is incomplete

## Installation

Requires:
- NeoForge 1.21.1
- GeckoLib

Download the latest jar from the Releases page.
